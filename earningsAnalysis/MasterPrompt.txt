Master Prompt — Earnings Briefing Style

Create a 10–15 min retail investor earnings briefing for [COMPANY] (NYSE/NASDAQ: [TICKER]) covering [QUARTER & YEAR] results released today [DATE]. Build as a single self-contained HTML file using Chart.js from cdnjs.
Typography: Bebas Neue for the masthead H1 (large, stacked, wide letter-spacing). DM Serif Display for section headings and KPI values. JetBrains Mono for all labels, metadata, timestamps, table headers, and chart axes. DM Sans weight 300 for body copy. Import all from Google Fonts.
Theme — Dark Mode: Background #060D1A, card surfaces #0E1E35, raised surfaces #122040. Accent colors: teal #26C6DA for section number labels, blue #42A5F5 for heading text, gold #FFB300 for highlights and milestones, green #00C853 for beats/positive, red #FF5252 for misses/negative. All borders #1A2E48. Body text #E8EDF5, secondary #A8B8CC, muted #5A7290.
Structure — in this order:

Masthead — dark navy with radial gradient glow, Bebas Neue giant H1, teal metadata line, colored badge row (green = beats, gold = milestones, blue = guidance, red = misses)
Milestone banner (if warranted) — gold left-border, dark raised card with icon + headline
KPI strip — 6-card grid, 1px gap border between cards, DM Serif values, mono labels, green change arrows
Transcript — timestamped sections at 00:00, 01:00, 02:30 etc. intervals. Blue italic blockquotes with executive quotes and cite attribution. Teal callout boxes for key investor insights. Max width 760px.
Table 1 — last 5 quarters: EPS estimate vs actual, revenue estimate vs actual, surprise delta, deliveries/key metric, stock reaction. Teal column headers on near-black header row. Alternating dark row shading. Current quarter highlighted blue-tint with left accent bar. Green/red beat/miss cells.
Table 2 — segment/product breakdown per quarter (revenue by division, margins, operating P&L, or equivalent)
6 Chart.js charts in this layout: two full-width bars (EPS vs estimate, Revenue vs estimate), then 2×2 grid (segment mix, key growth metric, profit/loss bar, YoY growth rate line). Dark grid #1A2E48, mono axis labels #5A7290, tooltips dark navy with border.
Stock performance chart — interactive tabs for 1W, 3M, 6M, 1Y. Stats bar showing start, current, period high/low, 52wk high/low, market cap, analyst target. Line color green if period positive, red if negative. Annotated event dots with legend strip. Populate with real anchored price data.
Bull case vs risks — two-column dark card layout, green/red arrow bullets
Guidance snapshot — 6-card grid, blue DM Serif values
Footer — dark navy, mono font, disclaimer, conversion rate if non-USD source data

All figures in USD. If source data is in another currency (RMB, EUR, KRW, etc.) convert at current rate and note the rate in the footer footnote.
Data requirements: Use real reported figures. For estimates, use analyst consensus (StreetAccount / LSEG). For stock price data, anchor to known real prices (52wk high/low, recent closes) and interpolate realistically between them. Note the earnings day reaction in the masthead badge.
